# Fecal DNA quantifies seabird diet

This repository contains data and code associated with the statistical analyses from the 
manuscript:

"DNA metabarcoding of feces provides quantitative measures of diet in two seabird species 
with contrasting foraging strategies" by Katelyn M. Depot, Cassandra Ciafro, Shannon Whelan,
Gemma V. Clucas, and Kyle H. Elliott
